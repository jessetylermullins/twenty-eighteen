LYG Redux

Code Louisville Front-End Web Development

HTML, CSS, JavaScript, jQuery, Typed.js, Adobe Illustrator

A bold redesign and rebranding of the Louisville Youth Group website, a local 501(c)3 nonprofit organization providing a variety of critical social services to queer and allied youth. The goal of this project was to convey a message of support and love to queer and allied youth, while also effectively communicating the services provided and the positive impact in the community over the past 25+ years.

This project was completed during a Code Louisville Front-End Web Development cohort, and uses a responsive layout that changes at custom breakpoints to adjust content when necessary, as well as JavaScript and jQuery to create interactive features like a responsive navigation and email validation.

A key challenge of this project was a limited timeframe. With several more projects' worth of experience, I would like to revisit this site and improve the code and functionality while maintaining the overarching design.
